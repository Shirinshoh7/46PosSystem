# 🧾 46POS — Система автоматизации розничной торговли

Профессиональное решение для автоматизации магазинов, аптек и малого бизнеса. Основной фокус проекта — высокая скорость работы интерфейса и полная автономность системы.

---

## 🛠 Технические особенности (Key Features)

* **Offline-First Architecture:** Система работает полностью локально. Отсутствие интернета не влияет на скорость обслуживания клиентов и стабильность базы данных.
* **Label & Receipt Printing:** Модуль интеграции с термопринтерами для печати чеков, ценников и штрих-кодов (Label Printing).
* **Inventory Management:** Учет остатков, приход/списание товаров и контроль критических запасов в реальном времени.
* **Smart Backup System:** Автоматизированное резервное копирование данных для предотвращения потери информации при сбоях оборудования.
* **Debt & Expense Tracking:** Модули учета дебиторской задолженности клиентов и контроля операционных расходов.

---

## 🏗 Технологический стек (Tech Stack)

* **Frontend:** Flutter (Dart) — кроссплатформенный интерфейс с высокой производительностью.
* **Database:** SQLite — легковесная и быстрая реляционная БД для локального хранения данных.
* **Business Logic:** Чистая архитектура с разделением ролей (Администратор/Кассир).
* **Hardware Integration:** Поддержка периферийных устройств (сканеры штрих-кодов, принтеры).
